TypeScript utility library for simple, fast and accessible color manipulation

Installation

npm install @prjctimg/huetiful

Or with yarn:

yarn add @prjctimg/huetiful

Or with bun:

bun add @prjctimg/huetiful

Quick Start

import { token, colors, hueshift } from "@prjctimg/huetiful";

// Parse a color to hex
token("cyan"); // "#00ffff"

// Get Tailwind colors
colors("red"); // ['#fef2f2', '#fee2e2', ...]

// Generate a hue-shifted palette
hueshift("#3e0000"); // ['#ffffe1', '#ffdca5', ...]

vangogh()

Generates palettes inspired by Vincent van Gogh's distinctive color palette. Supports different artistic periods and contrast levels. 

import { vangogh } from "@prjctimg/huetiful";

// Default: Arles period (vibrant yellows/blues)
vangogh("yellow");
// ['#003200ff', '#005a00ff', '#00845fff', '#00acccff', ...]

// Netherlands period (darker, muted)
vangogh("blue", { period: "netherlands" });

// Paris period (lighter, more saturated)
vangogh("green", { period: "paris" });

// Adjust contrast
vangogh("orange", { contrast: "subtle" });

Options

  • period - "netherlands" | "paris" | "arles" (default: "arles")
  • contrast - "subtle" | "medium" | "bold" (default: "bold")
  • num - number of colors (default: 6)

impressionist()

Generates palettes inspired by the Impressionist movement (Monet, Renoir). Features broken color technique and time-of-day lighting. 

import { impressionist } from "@prjctimg/huetiful";

// Default broken color technique
impressionist("green");
// ['#003285ff', '#00abffff', '#00cfaaff', ...]

// Optical mixing (pointillism style)
impressionist("blue", { technique: "optical" });

// Different times of day
impressionist("white", { timeOfDay: "morning" });
impressionist("white", { timeOfDay: "evening" });
impressionist("white", { timeOfDay: "dusk" });

Options

  • technique - "broken" | "optical" (default: "broken")
  • timeOfDay - "morning" | "noon" | "evening" | "dusk" (default: "noon")
  • blueShadows - boolean (default: true)
  • num - number of colors (default: 6)

picasso()

Generates palettes inspired by Pablo Picasso's Blue and Rose periods. 

import { picasso } from "@prjctimg/huetiful";

// Blue Period (melancholic)
picasso("blue", { period: "blue" });
// ['#000e1fff', '#001a29ff', '#161c3bff', ...]

// Rose Period (warm)
picasso("rose", { period: "rose" });
// ['#e9a29dff', '#ae5062ff', '#ce617cff', ...]

// Adjust melancholy intensity (0-1)
picasso("blue", { period: "blue", intensity: 0.9 });

Options

  • period - "blue" | "rose" (default: "blue")
  • intensity - 0-1 for Blue Period melancholy (default: 0.5)
  • num - number of colors (default: 6)

hueshift()

Creates a palette where hue shifts as lightness changes.

import { hueshift } from "@prjctimg/huetiful";

hueshift("#3e0000");
// ['#ffffe1', '#ffdca5', '#ca9a70', '#935c40', '#5c2418', '#3e0000', ...]

earthtone()

Creates a palette between an earth tone and any color.

import { earthtone } from "@prjctimg/huetiful";

earthtone("pink", { earthtones: "clay" });
// ['#6a5c52', '#8d746a', '#b38d86', '#d9a6a6', '#ffc0cb']

pastel()

Generates a random pastel variant.

import { pastel } from "@prjctimg/huetiful";

pastel("green");
// ['#a8d8a8', '#b8e0b0', '#c8e8c0', ...]

scheme()

Creates classic color schemes.

import { scheme } from "@prjctimg/huetiful";

scheme("blue", { kind: "triadic" });
// ['#0000ff', '#00ff00', '#ff0000']

scheme("blue", { kind: "complementary" });
// ['#0000ff', '#ffff00']

token()

Parses any recognizable color to the specified format.

import { token } from "@prjctimg/huetiful";

token("cyan"); // "#00ffff"
token("cyan", { kind: "obj", targetMode: "lch" });
token("cyan", { kind: "arr" });
token("cyan", { kind: "num" });

alpha()

Gets or sets the alpha channel of a color.

import { alpha } from "@prjctimg/huetiful";

alpha("#a1bd2f0d"); // 0.050980392156862744
alpha("b2c3f1", 0.5); // "#b2c3f180"

luminance()

Gets or sets the luminance of a color.

import { luminance } from "@prjctimg/huetiful";

luminance("#a1bd2f"); // 0.4417749513730954
luminance("#a1bd2f", 0.5);
