# Huetiful

TypeScript utility library for simple, fast and accessible color manipulation.

## Overview

Huetiful is a color manipulation library that uses color theory principles to provide functions for working with color as both individual tokens or collections. It depends on [Culori](https://culori.js.org/) for color manipulation.

## Features

- **Color Conversion** - Parse and convert between color spaces (RGB, LCH, LAB, HSV, etc.)
- **Palette Generators** - Create palettes from a single color using various algorithms
- **Collection Methods** - Sort, filter, and analyze color collections
- **Accessibility** - WCAG contrast checking and color blindness simulation
- **Artist-Inspired Palettes** - Generate palettes inspired by famous artists
